# Welcome to BrickUI

BrickUI is a modern Vue 3 component library built with the latest technologies:

- **Vue 3** - The progressive JavaScript framework
- **TypeScript** - Type-safe development
- **Pinia** - Intuitive state management
- **UnoCSS** - Instant on-demand atomic CSS engine
- **Storybook** - Interactive component documentation

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

Start Storybook:

```bash
npm run storybook
```

### Build

Build for production:

```bash
npm run build
```

Build Storybook:

```bash
npm run build-storybook
```

## Component Library

Our component library includes:

- **BrickButton** - Versatile button component with multiple variants
- **BrickCard** - Flexible card component with slots
- More components coming soon...

## Design System

BrickUI follows a consistent design system with:

- **Color Palette** - Primary, secondary, success, and danger colors
- **Typography** - Consistent font sizes and weights
- **Spacing** - Uniform spacing scale
- **Shadows** - Multiple shadow levels
- **Border Radius** - Consistent rounded corners

## State Management

The library uses Pinia for state management, providing:

- **Reactive stores** - Composition API based stores
- **TypeScript support** - Full type safety
- **DevTools integration** - Easy debugging

## Styling

BrickUI uses UnoCSS for styling, offering:

- **Atomic CSS** - Utility-first approach
- **On-demand generation** - Only generate used styles
- **Customizable** - Easy theme customization
- **Performance** - Minimal CSS bundle size
